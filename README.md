block-walker
============

simple block-walker of mtf 


how to build
============
Ubuntu:

	mkdir test
	cd test
	cmake ..
	make
Windows (VS):

	mkdir test
	cd test
	cmake ..
then blockwalker.sln can you find, build it.


how to test
===========

Ubuntu:

	./block-walker ../tester/Backup.bkf

Windows:

	block-walker ..\tester\Backup.bkf
